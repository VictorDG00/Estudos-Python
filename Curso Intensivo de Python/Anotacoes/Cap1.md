1.2 Erros de digitação em hello world, o que precisa fazer para gerar erro? Há alguma alteração que não gera erro?

Gerar erro: remover um dos parênteses gera erro explícito que até informa onde é o erro, assim como tentar usar aspas simples com dupla para fechar o texto não funciona.

Não gera erro: colocar ; no final não gera erro, o código roda igual, \n funciona para pular linha, {} muda a cor do texto mesmo que não mude nada no output.

2*print= não funciona, mas se fizer print=(2*"") ele imprime o texto 2 vezes.

Vírgula dentro do print concatena o texto com outras coisas como int ou string print('', 2), assim como + print(''+ 2).
